    使用此面板播放器的相關注意事項：

        您可以在設定播放清單項目的媒體標題時，使用像 &#12345; 這些"Unicode/萬國碼"
        來轉換成您想要的字符，包括繁體中文以外的雙字節字元 (例如韓文、日文) 和其他
        特殊字元；但您同時也需要將「在瀏覽器的狀態列顯示播放器文字訊息」的設定項目
        (blnStatusBar)設為 false，因為瀏覽器的狀態列是無法轉換像 &#12345; 這些萬國
        字符控制碼的。

  ▼2003.04.04: RUBY面板 (v1.0)

    由於這是此面板中文化後首個發表的版本，以下僅列出比對原始韓文版之間的差異：

     ※ 將大部份需要手動設定的原始碼內容，從播放面板 HTML 頁面檔案 (exobud.htm)
        匯出至幾個獨立檔案，讓使用者方便修改和管理。這些檔案包括：

            exobud.css   : ExoBUD MP 播放面板 CSS 樣式表
            exobudpl.css : ExoBUD MP 播放清單 CSS 樣式表
            exobudpl.js  : ExoBUD MP 播放清單設定檔 (JavaScript)
            exobudset.js : ExoBUD MP 播放程式基本設定檔 (JavaScript)

     ※ 播放清單的彈出式視窗，改為可調校視窗大小(resizable=yes)。

     ※ 變更部份圖檔名稱，使其更方便管理；原本以 .html 為副檔名的檔案，改為 .htm 。

     ※ 加入「面板製作資訊」及「更新異動」說明部份。

     ※ 由於此面板只適合用來播放音訊的媒體檔案，所以在這個版本已刪除了包含範例視訊
        檔案的資料夾 (sample) ，並在樣本播放清單內，以其他的媒體檔案連結來取代原有
        項目，同時額外增加６首驚直喜愛的『新世紀音樂』^^
